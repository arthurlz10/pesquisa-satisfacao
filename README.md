## 📌 Sobre o projeto  
Este é um projeto simples desenvolvido em Python, com o objetivo de realizar uma pesquisa de opinião sobre o atendimento ao cliente.

Ele foi criado para praticar o conceito de programação baseado no ciclo:  
➡️ Entrada → Processamento → Saída

---

## 🎯 Funcionalidades  
- Recebe o nome do entrevistado  
- Recebe a idade  
- Recebe a opinião sobre o atendimento (1: Excelente, 2: Bom, 3: Ruim)  
- Realiza múltiplas entrevistas utilizando repetição  
- Conta a quantidade de respostas de cada tipo  
- Exibe o resultado final da pesquisa  

---

## 🧠 Lógica utilizada  
- Uso de estrutura de repetição (`for`) para coletar dados de vários usuários  
- Uso de estruturas condicionais (`if/elif`) para classificar as respostas  
- Utilização de variáveis contadoras para armazenar os resultados  

---

## 🧪 Testes  
O programa foi testado inicialmente com uma quantidade menor de entrevistados para validar a lógica e, posteriormente, ajustado para 50 entrevistas conforme solicitado.
